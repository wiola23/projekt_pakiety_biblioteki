# projektpakiety

## Description
The project solves systems of linear equations in form of Ax=b, where A is a square matrix and b is a numeric vector. It uses a method of LU decomposition with partial pivoting and is written in C.

## Installation
In order to install the project from GitHub:
remotes::install_github("wiola23/projektpakiety")
